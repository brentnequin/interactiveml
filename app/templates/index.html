About

Interactive ML is a Python-based web tool designed to visualize machine learning algorithms.

Currently, Interactive ML performs unsupervised algorithms. Plotly was used to produce the graph along with the plotted data.

This web tool is developed by Brent Nequin.

×

How To Use

1. Select the number of clusters.

2. Place training points and testing points on the graph by aiming your cursor on positions on the graph and clicking. Choose between training points and testing points within the menu.

3. Run a clustering algorithm by clicking on the algorithm buttons.

The graph will update by showing the cluster centers and resulting labels of the clustering algorithm. Notice the training points in each cluster have their own color. The testing points will also take the same color of the nearest cluster meaning that it belongs to that cluster.

To hide a trace, click on the trace to be hiden on the legend.

To show only one trace, double click on the trace to be shown on the legend.
